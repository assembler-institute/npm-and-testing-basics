# NPM PILL<br>

**Initialize NPM into your project**

`npm init -y`

**Install Moment.js**

`npm install moment`

**Install Lodash.js**

`npm install lodash@4.17.14`

**Install jQuery ( Production )**

` npm install --save jquery`

**Install Jest ( Development )**

`npm install --save-dev jest`

**Update NPM**

`npm install -g npm@latest`
